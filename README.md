# Chess
Build a Chess game in C++

Overview of requirements:

__Chess board:__ 
  - Size 8x8 
  - Black/white squares 
  - King starts on same color
    
__Piece Count:__
  - Pawns:   8 
  - Rooks:   2 
  - Knights: 2 
  - Bishops: 2 
  - Queens:  1 
  - Kings:   1

Player: Black pieces/white pieces

No colisions 

Check for check/checkmate 

Pawn promotion/en passat

Look for GUI

__Functions Needed:__
  - Map
  - Players
  - Pieces
  - Check(mate) test
  - GUI
