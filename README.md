# Chess
Build a Chess game in C++

Overview of requirements:

Chess board: 
  
    Size 8x8 
  
    Black/white squares 
  
    king starts on same color
    
Piece Count:
      Pawns: 8 
      Rooks: 2 
      Knights: 2 
      Bishops: 2 
      Queens: 1 
      Kings: 1
Player: Black pieces/white pieces
No colisions 
Check for check/checkmate 
Pawn promotion/en passat
Look for GUI

Functions Needed:
  Map
  Players
  Pieces
  Check(mate) test
  GUI
